# 链表基础操作

链表末尾添加节点

```ts
function addNode(listA, listB) {
  let curA = listA;
  // 首先转到当前链表末尾
  while (curA.next) {
    curA = curA.next;
  }
  // 向末尾添加
  curA.next = listB;
}
```

删除某个节点

```ts
function removeNode(list, target) {
  const dummy = { next: list };
  let temp = dummy;
  while (temp.next) {
    if (temp.next.val === target) {
      temp.next = temp.next.next;
    } else {
      temp = temp.next;
    }
  }
}
```
